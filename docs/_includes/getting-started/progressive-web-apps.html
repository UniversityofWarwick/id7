Progressive Web Apps

We've included icons to help you set up your application as a Progressive Web App (PWA) if you want. You'll find them in the images folder with the form pwa-icon-[size]x[size].png.

Here's an example of a PWA manifest.json file you can update and use for your application:

{
  "short_name": "ID7",
  "name": "ID7 - University of Warwick brand",
  "icons": [
    {
      "src": "/dist/images/pwa-icon-512x512.png",
      "type": "image/png",
      "sizes": "512x512",
      "purpose": "any maskable"
    },
    {
      "src": "/dist/images/pwa-icon-192x192.png",
      "type": "image/png",
      "sizes": "192x192",
      "purpose": "any maskable"
    }
  ],
  "id": "/",
  "start_url": "/",
  "background_color": "#A499F4",
  "display": "standalone",
  "scope": "/",
  "theme_color": "#A499F4",
  "description": "Standalone app for University of Warwick's ID7 brand",
}

There's more you can do with PWAs if you're intending for people to actually install your app. See web.dev's guidelines for more.
