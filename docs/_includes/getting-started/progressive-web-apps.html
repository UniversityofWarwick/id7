Progressive Web Apps / Web Manifests

We recommend declaring a web manifest in your template - it defines some extra icons and other descriptive properties, which help the app look better when shown as an icon.

Depending on your needs, you may also use this manifest to declare your app as a Progressive Web App - that is optional and you can find more information in web.dev's guidelines.

This is a minimal example to use the official icons - you may want to use custom icons and/or adjust the name, description and colour properties. 

{
  "short_name": "Warwick",
  "name": "University of Warwick",
  "icons": [
    {
      "src": "/dist/images/android-chrome-512x512.png",
      "type": "image/png",
      "sizes": "512x512",
      "purpose": "any"
    },
    {
      "src": "/dist/images/pwa-icon-512x512.png",
      "type": "image/png",
      "sizes": "512x512",
      "purpose": "maskable"
    }
  ],
  "id": "/",
  "start_url": "/",
  "background_color": "#A499F4",
  "display": "standalone",
  "scope": "/",
  "theme_color": "#A499F4",
}
